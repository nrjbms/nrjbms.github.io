Reservation App




DESCRIPTION

This application was made using Android Studio using thier features of EditText, DataPicker and TimePicker. It is suppose to help restaurants store reservations for their use. This was one of the first projects I made using Java language and android studio to create a working application

COMMENTS

During the coding process, I learnt about how to use the features in the Android Studio and linked it to codes and how to get it working as an application. Despite my limited knowledge, I was able to make do with the information given and created an app that might help restaurants in the future. Looking back on it now, I think there is alot to be improved on the app but I still think it was a good challenge to understand the coding and grow my skills on android development.
